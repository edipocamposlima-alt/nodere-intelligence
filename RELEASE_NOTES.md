# Release Notes - Nodere Intelligence MVP

## Status

Liberado para uso local em MVP pessoal.

## Versao

`0.1.0-local-mvp`

## Incluido nesta entrega

- Aplicativo web responsivo em `index.html`.
- Identidade visual Nodere com novo logo.
- PWA basico com manifest e service worker.
- Dashboard executivo.
- Buscador inteligente.
- Google Intelligence.
- Tela de empresa com diagnostico.
- CRM Kanban com etapa comercial.
- Historico comercial e notas.
- Automacao e mensagens comerciais.
- Inbox comercial simulado.
- Copilot IA simulado.
- Relatorios com CSV e PDF via impressao.
- Admin com usuarios e permissao por aba.
- Area de configuracao da API MVP.
- Backend Node MVP em `backend/`.
- Schema Supabase enxuto em `mvp-supabase-schema.sql`.

## Validado

- Navegacao desktop.
- Navegacao mobile.
- Novo logo em desktop e mobile.
- Sem overflow horizontal mobile.
- Sintaxe de frontend.
- Sintaxe de backend.
- Controles de CRM e historico.

## Limitacoes conhecidas

- Login ainda e demonstrativo/local.
- Dados reais dependem de Supabase, Google Places e OpenAI configurados.
- WhatsApp usa link direto, nao Cloud API.
- Multiusuario real exige autenticacao e enforcement no backend.

- Jobs recorrentes de busca.
- WhatsApp Cloud API.
